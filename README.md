# FitLife Pro

A full-stack fitness tracking application built with:

- React
- Node.js
- Express
- PostgreSQL
- Docker
- Jenkins
- Kubernetes
- Prometheus
- Grafana

## Features

- User Authentication
- Workout Tracking
- Meal Tracking
- Water Tracking
- Weight Tracking
- Dashboard
- CI/CD Pipeline
- Monitoring
